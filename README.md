<h1 align='center'>Fitness Journal</h1>

### Install dependencies.

```
npm ci
```

### Start app.
```
npm run app:start
```

### Start app on android.

```
npm run app:android
```

### Start app on ios.

```
npm run app:ios
```

### Start app on web.

```
npm run app:web
```

### Run app tests.

```
npm run app:test
```

### Sort app translations keys.

```
npm run app:sort-json
```

### Start api.

```
npm run api:start
```

### Run api test.

```
npm run api:test
```

### test code.

```
npm run test
```

### Lint code.

```
npm run lint
```
