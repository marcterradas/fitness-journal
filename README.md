<h1 align='center'>Fitness Journal</h1>

### Install dependencies.

```
npm ci
```

### Start app.
```
npm app:run start
```

### Start app on android.

```
npm app:run android
```

### Start app on ios.

```
npm app:run ios
```

### Start app on web.

```
npm app:run web
```

### Run app tests.

```
npm app:run test
```

### Sort app translations keys.

```
app:sort-json
```

### Start api.

```
npm api:run start
```

### Run api test.

```
npm api:run test
```

### test code.

```
npm run test
```

### Lint code.

```
npm run lint
```
