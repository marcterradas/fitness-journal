<h1 align='center'>Fitness Journal - APP</h1>

### Start expo server.

```
npm run start
```

### Compiles and hot-reloads for development (Web).

```
npm run web
```

### Compiles and hot-reloads for development (Android).

```
npm run android
```

### Compiles and hot-reloads for development (iOS).

```
npm run ios
```

### Run unit tests.

```
npm run test
```

### Sort translations keys.
```
npm run sort-json
```
